# Changelog — Маяк VPN (Android)

Версионирование приложения Маяк ведётся ОТДЕЛЬНО от версии движка AmneziaWG (форк-база).
`versionName` — продукт Маяка; `versionCode` монотонно растёт (для установки апдейтов поверх).
Версия движка показывается в «О приложении» отдельно. Формат: [SemVer](https://semver.org).

> ⚠️ Замечание по версионированию: с `versionCode 13` имя `0.1.1` не двигали, хотя
> `versionCode` дорос до 22. Сборки 14–22 — поток debug-билдов под одним именем 0.1.1.
> На будущее: поднимать `versionName` при заметных изменениях, чтобы релизы были различимы.
> Текущее: `gradle.properties` — `mayakVersionName=0.1.1`, `mayakVersionCode=22`.

## [Поток 0.1.1] — versionCode 14–22 (2026-06-28)

Девять debug-сборок за день под именем 0.1.1 (`versionName` не поднимали). Главное —
обход блокировки нашего домена DNS-ом оператора через DoH и устойчивость подключения.

### Добавлено
- **DoH-резолв endpoint** — имя экзита резолвится через шифрованный DNS (DoH) мимо DNS
  оператора, с IP-фоллбэком (vc18). Корень мобильного бага: оператор резал наш домен по DNS.
- **DoH-мультипровайдер** — Cloudflare + Google + Quad9, все на 443, для стойкости к блоку
  одного провайдера (vc19). Подтверждено на сотовой Билайна (build 21).
- **Кнопка подключения = тоггл с ОТМЕНОЙ** во время подключения (vc14).
- **Авто-обновление списка стран + кнопка «Обновить»** — новые направления без перелогина (vc22).
- **Диагностика**: «Диагностика (логи)» в настройках со встроенным лог-вьюером и отправкой
  диаг-лога на сервер; настройки на Material 3.

### Изменено
- **Движок: amneziawg-go → v0.2.19** (реально последняя, с S3/S4) — откат с ошибочного v1.0.4 (vc15).
- Предзагрузка конфига в кэш: не дёргать `api.mayakvpn.ru` в момент коннекта (vc17).

### Исправлено / Безопасность
- DoH-резолв вынесен на `Dispatchers.IO` (H1); форс HTTPS в `MayakBackend` (L2) (vc20).
- M3: диаг-лог собирает только наши логи и вычищает секреты; M4: кэш конфига одноразовый (vc21).

### Тех./диагностика
- Временный зашитый конфиг к экзиту (без `/connect`) для диагностики мобильного DPI (vc16) —
  диагностическая сборка.

## 0.1.0 (2026-06-27) — versionCode 12

Первая сборка с СОБСТВЕННОЙ версией Маяка (раньше приложение показывало версию движка
AmneziaWG `2.0.1` как свою — это исправлено).

- Введено собственное версионирование приложения (`mayakVersionName`/`mayakVersionCode`),
  отдельно от версии движка AmneziaWG.
- D1: кэш направлений — смена темы больше НЕ дёргает сеть (рефетч конфигов только по делу).
- Стабильный HWID при регистрации устройства (переустановка не плодит устройства).
- Момент подключения как событие + анимированный сетевой фон (DESIGN-VISION).
- Дизайн-доводка: движение круга подключения, glow, токены, флаги, переходы.
- Движок: AmneziaWG 2.0.1 (userspace), AWG 2.0-обфускация.

## Пред-релизная разработка (до собственного версионирования) — 2026-06-25…26

Работа шла под версией движка AmneziaWG (отдельного `versionName` ещё не было). Хронология
по git-истории форка [wobujidao/amneziawg-android](https://github.com/wobujidao/amneziawg-android):

### 2026-06-26
- Анимированный сетевой фон и подключение «как событие» (DESIGN-VISION); дизайн-доводка
  движения круга, glow, токенов, флагов и переходов.
- Стабильный HWID в регистрации устройства (app+core).

### 2026-06-25
- **Ребрендинг «Маяк VPN»**: `app_name = "Маяк VPN"`, реальный логотип-маяк (экран + иконка
  лаунчера), брендовый редизайн (Material 3, фирменные цвета).
- **i18n (7 языков)**: ru/be/kk/uz/en/de/fr + выбор языка в приложении.
- **Главный экран в стиле Happ**: круглая кнопка подключения + статус + таймер сессии + список
  стран с флагами; авто-повтор пробы скрывает 15-сек привязку.
- **DayNight-тема** (авто по системе + ручной выбор), переключатель на всех экранах; экраны
  «Настройки» и «О приложении» (версия, основа AmneziaWG/движок/AWG 2.0, лицензия, OSS).
- **Вход по email** + домен `api.mayakvpn.ru` с IP-фоллбэком; ссылки на кабинет и правовые
  документы.
- Импорт регистрации: «Сканировать QR» (zxing) + «Вставить рег-ссылку»
  (`mayak://reg?server=&login=&password=`).
- Зашитый CA → ECDSA P-256 (Android не поддерживает Ed25519 TLS); стабильная debug-подпись;
  фикс S3/S4=0 под userspace-движок ядра.
- **`:core`** — переносимый не-UI-модуль (чистый Kotlin/JVM): `MayakBackend` (HTTP-клиент к API +
  фейловер доменов), `ConfRenderer` (JSON→.conf), DTO 1:1 с Go.
- **Бета-экран (T3+T4)**: вход → страны → подключение поверх `:core`, авто-резерв.
- **T1 — форк + baseline**: создан форк `wobujidao/amneziawg-android` (+upstream), собран
  baseline debug-APK (~13 МБ) на GitHub Actions (хост чист).

> Сборки: debug-APK с постоянной debug-подписью (апдейты ставятся поверх предыдущих).
> Базовая совместимость: Android 8+ (как у форк-базы AmneziaWG).
> База форка: AmneziaWG-android (upstream), движок amneziawg-go (userspace, AWG 2.0).
